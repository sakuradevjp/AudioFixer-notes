## What is AudioFixer?

A Windows app that normalizes or boosts the audio of any video file **without re-encoding the video stream**. Drag, drop, done — quality stays bit-for-bit identical.

- 🎚️ Auto Normalize (EBU R128) or manual boost up to 1000%
- 🎬 Lossless: only the audio track is processed
- 📦 Drag & drop, batch supported
- 🔒 100% local processing, no uploads
- 📁 MP4, MKV, MOV, AVI, MP3, WAV, FLAC, AAC

**Get it on Microsoft Store**: https://apps.microsoft.com/detail/9nkn2q9svrs1
